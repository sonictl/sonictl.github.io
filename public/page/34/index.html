重新指派usb转串口模块在linux系统中的设备调用名称

Command: >> lsusb This lists your usb devices and "067b:2303",for instance, as its ID or: >> ls -l /sys/bus/usb-serial/devices An other locations worth exploring are /sys/class/tty/ Write this to /etc/udev/rules.d/50-usb.rules SUBSYSTEM=="tty", ATTRS{idVendor}=="19d2", ATTRS{idProduct}=="0031", SYMLINK+="ONDA" SUBSYSTEM=="tty", ATTRS{idVendor}=="12d1", ATTRS{idProduct}=="14ac", SYMLINK+="HUAWEI" Reboot. Now you can use /dev/ONDA to get to your ONDA device. If you want non-root users to be able to use these, then add , MODE="0666" to the end of each line.
