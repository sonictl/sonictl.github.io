hist_sig_dict = {}
gender_dict = {}
birth_dict = {}

Now we can loop through our nodes list and add the appropriate items to each dictionary.

for node in nodes: # Loop through the list, one row at a time

Now you have a set of dictionaries that you can use to add attributes to nodes in your Graph object. The set_node_attributes еҮҪж•°жңү3дёӘеҸҳйҮҸпјҡ

  • the Graph to which youвҖҷre adding the attribute,
  • the name of the new attribute.
  • the dictionary of id-attribute pairs.

triadic closure#

Triadic closure жҳҜжҢҮеҰӮжһң2дәәжӯЈеҘҪи®ӨиҜҶеҗҢ1дёӘдәәпјҢйӮЈд№Ҳиҝҷ2дёӘдәәеҖҫеҗ‘дәҺдә’зӣёи®ӨиҜҶгҖӮиҝҷеӣһйҖ жҲҗдёүи§’еҪўзҡ„й—ӯзҺҜпјҢcompleting a triangle in the visualization of three edges connecting Fox, Fell, and Whitehead. The number of these enclosed triangles in the network can be used to find clusters and communities of individuals that all know each other fairly well.

и®Ўз®—triadic closureзҡ„ж–№жі•еҸ«clustering coefficientпјҢbecause of this clustering tendency. дҪҶеңЁиҝҷйҮҢиҰҒз”Ёзҡ„жҰӮеҝөеҸ«Transitivity. Transitivity is the ratio of all triangles over all possible triangles.

You can calculate transitivity in one line, the same way you calculated density:

triadic_closure = nx.transitivity(G)
print("Triadic closure:", triadic_closure)

QuakerзҪ‘з»ңдёӯзҡ„transitivity = 0.1694е·ҰеҸіпјҢжҜ”density 0.0248 еӨ§пјҢиҜҙжҳҺеӣҫдёҚзЁ еҜҶпјҢиҝҳжңүдёҖдәӣpossible triangles to begin with. д№ҹиҜҙжҳҺпјҢйӮЈдәӣжңүжӣҙй«ҳеәҰж•°зҡ„nodesжӣҙеҖҫеҗ‘дәҺеӯҳеңЁеңЁдёүи§’еҪўдёӯгҖӮд№ҹе°ұжҳҜиҜҙпјҢе·Із»ҸжңүеҫҲеӨҡиҝһжҺҘзҡ„иҠӮзӮ№еҫҲеҸҜиғҪжҳҜиҝҷдәӣе°Ғй—ӯдёүи§’еҪўзҡ„дёҖйғЁеҲҶгҖӮдёәдәҶж”ҜжҢҒиҝҷдёҖзӮ№пјҢжӮЁйңҖиҰҒжӣҙеӨҡең°жіЁж„Ҹе…·жңүй«ҳеәҰж•°зҡ„иҠӮзӮ№гҖӮ a good next step is to find which nodes are the most important ones in your network.

Centrality#

Which nodes are the most important?вҖқ there are many different ways of calculating centrality. Here youвҖҷll learn about three of the most common centrality measures: degree, betweenness centrality, and eigenvector centrality.

еәҰж•°degree

  • жңҖз®ҖеҚ•зӣҙжҺҘеёёз”Ёзҡ„дёҖдёӘз»ҹи®ЎйҮҸпјҢе°ұжҳҜдёҖдёӘиҠӮзӮ№иә«дёҠзҡ„иҫ№зҡ„ж•°йҮҸгҖӮ е…ҲеҲӣе»әдёҖдёӘеҢ…еҗ«degreeзҡ„dictionaryпјҢдҪңдёәattribute ж·»еҠ еҲ°жҜҸдёӘиҠӮзӮ№дёӯгҖӮThe keys are nodes and the values are centrality measures.
degree_dict = dict(G.degree(G.nodes()))
nx.set_node_attributes(G, 'degree', degree_dict) #жіЁж„ҸеҸӮж•°йЎәеәҸ
print(G.node['William Penn']) #see William PennвҖҷs degree along with his other information

еңЁдҪ зҪ‘з»ңзҡ„жүҖжңүиҠӮзӮ№G.nodes()зҡ„еҲ—иЎЁдёҠпјҢдҪҝз”ЁG.degree()ж–№жі•гҖӮеҸҜзңӢеҲ°consoleиҫ“еҮәпјҡ

еҰӮжһңдҪ жғіз”»еҮә degree зҡ„еҲҶеёғзӣҙж–№еӣҫпјҢеҸӮиҖғпјҡdraw histogram

еңЁpythonеҸҜд»Ҙд»»ж„ҸsortдёҖдәӣеҖј: жҜ”еҰӮдҪҝз”ЁеҶ…е»әзҡ„sourted()еҮҪж•°жқҘsortдёҖдёӘdictionary by its keys or values. е№¶жүҫеҲ°top 20зҡ„nodes ranked by degree. дҪ йңҖиҰҒз”Ёitemgetter()жқҘеё®дҪ жҗһе®ҡгҖӮеҰӮдёӢпјҡ

е…¶е®һдҪ зңӢеҲ°зҡ„еәҰж•°й«ҳзҡ„дәәпјҢйӮЈдәӣhubпјҢеҰӮWilliam Penn, д»–жҳҜ Quaker leader зЎ®е®һеҫҲйҮҚиҰҒпјҢдҪҶеӨҡж•°social networks have just few hubs, еү©дёӢзҡ„дёҚйӮЈд№Ҳй«ҳзҡ„еҸҲжҖҺж ·е‘ўпјҹ еҫҲеӨҡж—¶еҖҷйӮЈдәӣй«ҳеәҰж•°зҡ„иҠӮзӮ№еёҰжқҘзҡ„дҝЎжҒҜе№¶дёҚд»ӨдәәжғҠеҘҮпјҢжҜ”еҰӮ Penn or Quakerism co-founder Margaret Fell, with a degree of 13. дҪ дјҡеҸ‘зҺ°й«ҳеәҰж•°зҡ„дәәеҹәжң¬дёҠе°ұжҳҜйӮЈдәӣfounder.

зү№еҫҒеҗ‘йҮҸдёӯеҝғжҖ§ Eigenvector centrality is a kind of extension of degree - е®ғиҖғиҷ‘дәҶжң¬иә«зҡ„еәҰж•°е’Ңе®ғйӮ»еұ…зҡ„еәҰж•°пјҲдҪҶдёҚжҳҜз®ҖеҚ•зҡ„жҜ”дҫӢе…ізі»пјҢиҰҒз”Ё$Ax = \lambda x$з®—пјүгҖӮEigenvector centrality cares if you are a hub, but it also cares how many hubs you are connected to. еҜ№дәҺдәҶи§Је“ӘдәӣиҠӮзӮ№еҸҜд»Ҙеҝ«йҖҹиҺ·еҸ–е…¶д»–иҠӮзӮ№зҡ„дҝЎжҒҜпјҢEigenvector centralityе°ұжңүз”ЁдәҶгҖӮе®ғиў«и®Ўз®—дёәд»Һ0еҲ°1зҡ„еҖјпјҡи¶ҠжҺҘиҝ‘1пјҢдёӯй—ҙеәҰи¶ҠеӨ§гҖӮеҰӮжһңдҪ и®ӨиҜҶеҫҲеӨҡжңүе…ізі»зҡ„дәәпјҢдҪ еҸҜд»ҘеҫҲжңүж•Ҳең°дј ж’ӯдҝЎжҒҜгҖӮ

д»Ӣж•°дёӯеҝғжҖ§ Betweenness centrality is a bit different. д»Ӣж•°дёӯеҝғжҖ§ doesnвҖҷt care about the number of edges any one node. д»Ӣж•°дёӯеҝғжҖ§иҖғиҷ‘зҡ„жҳҜжүҖжңүзҡ„shortest paths that pass through a particular node. To do this, it must first calculate every possible shortest path in your network, so keep in mind that betweenness centrality will take longer to calculate than other centrality measures. д»Ӣж•°дёӯеҝғжҖ§(д№ҹд»Ҙ0еҲ°1зҡ„е°әеәҰиЎЁзӨә)еңЁжҹҘжүҫиҝһжҺҘзҪ‘з»ңдёӯдёӨдёӘдёҚеҗҢйғЁеҲҶзҡ„иҠӮзӮ№ж–№йқўзӣёеҪ“еҮәиүІгҖӮеҰӮжһңжӮЁжҳҜиҝһжҺҘдёӨдёӘйӣҶзҫӨзҡ„е”ҜдёҖеҜ№иұЎпјҢйӮЈд№ҲиҝҷдәӣйӣҶзҫӨд№Ӣй—ҙзҡ„жүҖжңүйҖҡдҝЎйғҪеҝ…йЎ»з»ҸиҝҮжӮЁгҖӮдёҺhubдёҚеҗҢпјҢиҝҷз§ҚиҠӮзӮ№йҖҡеёёз§°дёәbrokerд»ЈзҗҶгҖӮиҝҷдёӘжҢҮж ҮдёҚжҳҜжүҫеҲ°brokerageзҡ„е”ҜдёҖж–№жі•пјҢиҝҳжңүеҲ«зҡ„зі»з»ҹеҢ–зҡ„ж–№жі•гҖӮдҪҶе®ғеҫҲз®ҖеҚ•пјҢдёҚжҳҜеӣ дёәе®ғеҚ жңүиҫ№ж•°еӨҡпјҢиҖҢжҳҜе®ғжҸҗдҫӣдәҶдёҖдёӘдёҚеҗҢgroupsд№Ӣй—ҙзҡ„жЎҘжўҒпјҢcohesion.

These two centrality measures are even simpler to run than degreeвҖ”they donвҖҷt need to be fed a list of nodes, just the graph G. You can run them with these functions:

eigenvector_dict = nx.eigenvector_centrality(G) # Run eigenvector centrality
betweenness_dict = nx.betweenness_centrality(G) # Run betweenness centrality

# Assign each to an attribute in your network
nx.set_node_attributes(G, 'betweenness', betweenness_dict)
nx.set_node_attributes(G, 'eigenvector', eigenvector_dict)

еҗҢж ·еҸҜд»Ҙsort иҝҷдәӣйҮҸпјҡ

sorted_betweenness = sorted(betweenness_dict.items(), key=itemgetter(1), reverse=True)

print("Top 20 nodes by betweenness centrality:")

Advanced NetworkX: Community detection with modularity е…ідәҺзҪ‘з»ңж•°жҚ®йӣҶзҡ„еҸҰдёҖдёӘеёёи§Ғй—®йўҳжҳҜпјҢеңЁжӣҙеӨ§зҡ„зӨҫдјҡз»“жһ„дёӯпјҢеӯҗз»„жҲ–зӨҫеҢәжҳҜд»Җд№ҲгҖӮдҪ зҡ„зӨҫдәӨзҪ‘з»ңжҳҜдёҖдёӘдәәдәәйғҪи®ӨиҜҶзҡ„еҝ«д№җеӨ§е®¶еәӯеҗ—?жҲ–иҖ…пјҢе®ғжҳҜдёҖз»„иҫғе°Ҹзҡ„еӯҗз»„пјҢд»…з”ұдёҖдёӨдёӘдёӯд»ӢиҝһжҺҘ? жңүеҘҪеӨҡж–№жі•пјҢдҪҶжҜ”иҫғеёёи§Ғзҡ„жҳҜз”ЁmodularityжқҘеҒҡзӨҫеҢәеҸ‘зҺ°гҖӮModularity is a measure of relative density in your network: a community (called a module or modularity class) has high density relative to other nodes within its module but low density with those outside.

Modularity gives you an overall score of how fractious your network is, and that score can be used to partition the network and return the individual communities.

Though we wonвҖҷt cover it in this tutorial, itвҖҷs usually a good idea to get the global modularity score first to determine whether youвҖҷll learn anything by partitioning your network according to modularity. To see the overall modularity score, take the communities you calculated with communities = community.best_partition(G) and run global_modularity = community.modularity(communities, G). Then just print(global_modularity).

Community detection and partitioning in NetworkX requires a little more setup than some of the other metrics. There are some built-in approaches to community detection (like minimum cut, but modularity is not included with NetworkX. Fortunately thereвҖҷs an additional python module you can use with NetworkX, which you already installed and imported at the beginning of this tutorial. You can read the full documentation for all of the functions the community module offers, but for most community detection purposes youвҖҷll only want best_partition():

communities = community.best_partition(G)

дёҠйқўд»Јз ҒдјҡеҲӣе»әдёҖдёӘеӯ—е…ёпјҢbest_partition()иҰҒзңӢйҖӮеҗҲиҝҷдёӘgraphзҡ„жңҖеҗҲзҗҶзҡ„# of communities. е№¶з»ҷжҜҸдёӘnode assignдёҖдёӘзј–еҸ·пјҢд»Һ0ејҖе§ӢгҖӮе°ұжҳҜжҜҸдёӘnode belong to зҡ„зј–еҸ·гҖӮдҪ еҸҜд»Ҙadd these values to nodes in the same way:

еңЁеғҸиҝҷж ·зҡ„е°ҸеһӢзҪ‘з»ңдёӯпјҢдёҖдёӘеёёи§Ғзҡ„д»»еҠЎжҳҜжҹҘжүҫе№¶еҲ—еҮәжүҖжңүжЁЎеқ—еҢ–зұ»еҸҠе…¶жҲҗе‘ҳгҖӮеңЁиҫғеӨ§зҡ„зҪ‘з»ңдёӯпјҢеҲ—иЎЁеҸҜиғҪй•ҝеҫ—йҡҫд»Ҙйҳ…иҜ»пјҢеҸҜд»ҘйҖҡиҝҮеҸҜи§ҶеҢ–зҪ‘з»ңе№¶ж №жҚ®иҠӮзӮ№зҡ„жЁЎеқ—зұ»дёәе…¶ж·»еҠ йўңиүІгҖӮYou can do this by manipulating the communities dictionary. еңЁз”ЁPythonеӨ„зҗҶе’Ңе®ЎжҹҘдәҶдёҖзі»еҲ—зҪ‘з»ңжҢҮж Үд№ӢеҗҺпјҢжӮЁзҺ°еңЁжңүдәҶе…ідәҺиҝҷдёӘж—©жңҹзҺ°д»ЈиӢұеӣҪж•ҷеҸӢжҙҫдҝЎеҫ’зҪ‘з»ңзҡ„дәүи®әе’Ңз»“и®әзҡ„иҜҒжҚ®гҖӮдҫӢеҰӮпјҢжӮЁзҹҘйҒ“зҪ‘з»ңзҡ„еҜҶеәҰзӣёеҜ№иҫғдҪҺпјҢиҝҷж„Ҹе‘ізқҖжқҫж•Јзҡ„е…іиҒ”е’Ң/жҲ–дёҚе®Ңж•ҙзҡ„еҺҹе§Ӣж•°жҚ®гҖӮдҪ зҹҘйҒ“пјҢиҝҷдёӘзӨҫеҢәжҳҜеӣҙз»•зқҖеҮ дёӘдёҚжҲҗжҜ”дҫӢзҡ„еӨ§дёӯеҝғhubsз»„з»Үиө·жқҘзҡ„пјҢе…¶дёӯеҢ…жӢ¬зҺӣж јдёҪзү№В·иҙ№е°”(Margaret Fell)е’Ңд№”жІ»В·зҰҸе…Ӣж–Ҝ(George Fox)зӯүж•ҷжҙҫзҡ„еҲӣе§ӢдәәпјҢд»ҘеҸҠеЁҒе»үВ·дҪ©жҒ©(William Penn)зӯүйҮҚиҰҒзҡ„ж”ҝжІ»е’Ңе®—ж•ҷйўҶиў–гҖӮжӣҙжңүеё®еҠ©зҡ„жҳҜпјҢдҪ зҹҘйҒ“дёҖдәӣеӯҰеҺҶзӣёеҜ№иҫғдҪҺзҡ„еҘіжҖ§пјҢжҜ”еҰӮдјҠдёҪиҺҺзҷҪвҖўйҮҢз»ҙж–Ҝ(Elizabeth Leavens)е’ҢзҺӣдёҪвҖўдҪ©е®ҒйЎҝ(Mary Penington)пјҢеҘ№д»¬(з”ұдәҺдёӯй—ҙеәҰиҫғй«ҳ)еҸҜиғҪе……еҪ“дәҶдёӯй—ҙдәәbrokersпјҢе°ҶеӨҡдёӘзҫӨдҪ“иҒ”зі»иө·жқҘгҖӮжңҖеҗҺпјҢжӮЁдәҶи§ЈеҲ°зҪ‘з»ңжҳҜз”ұдёҖдёӘеӨ§з»„д»¶componentsе’Ңи®ёеӨҡйқһеёёе°Ҹзҡ„з»„д»¶з»„жҲҗзҡ„гҖӮеңЁиҝҷдёӘжңҖеӨ§зҡ„з»„жҲҗйғЁеҲҶдёӯпјҢжңүеҮ дёӘжҲӘз„¶дёҚеҗҢзҡ„зӨҫеҢәcommunitiesпјҢе…¶дёӯдёҖдәӣдјјд№ҺжҳҜеӣҙз»•ж—¶й—ҙжҲ–ең°зӮ№з»„з»Үиө·жқҘзҡ„(жҜ”еҰӮдҪ©жҒ©е’Ңд»–зҡ„зҫҺеӣҪеҗҢдәӢ)гҖӮз”ұдәҺжӮЁе°Ҷе…ғж•°жҚ®ж·»еҠ еҲ°зҪ‘з»ңдёӯпјҢеӣ жӯӨжӮЁеҸҜд»ҘдҪҝз”Ёе·Ҙе…·иҝӣдёҖжӯҘз ”з©¶иҝҷдәӣжҢҮж ҮпјҢе№¶еҸҜиғҪи§ЈйҮҠжӮЁзЎ®е®ҡзҡ„дёҖдәӣз»“жһ„зү№жҖ§гҖӮ
